## Petpet Generator

灵感来源魔性的petpet摸头gif图片

![demo](https://raw.githubusercontent.com/B1gM8c/Petpet/main/demo.gif)





## 功能介绍

- 支持PC端、移动端（部分浏览器可能不兼容）
- 支持白天、黑夜两种主题风格
- 支持对图片进行细节处理
  - 调整镜像
  - 调整尺寸
  - 调整挤压程度
  - 调整速度
